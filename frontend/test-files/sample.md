# EspressoBot Documentation

## Overview
EspressoBot is an AI-powered assistant for managing e-commerce operations.

## Features
- Product management
- Inventory tracking
- Price updates
- Customer support

## Usage
1. Upload product data
2. Process with AI agent
3. Review and approve changes

### Code Example
```javascript
const product = {
  name: "Espresso Machine",
  price: 699.99,
  sku: "ESP-001"
};
```
